# Enterprise Nexus Portfolio

Futuristic SAP Command Center portfolio for **Rishik Maduri**.

## Quick Start

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (typically `http://localhost:5173`).

## Build for Production

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to any static host (Netlify, Vercel, GitHub Pages, etc.).

## Features

- System boot startup animation
- Mission Control tree navigation
- Interactive technology network graph
- Digital verification chamber for SAP certification
- Explorable enterprise project modules
- Career timeline & achievement badges
- Mouse-reactive holographic UI
- Cinematic panel transitions

## Tech Stack

- Vite
- Vanilla HTML / CSS / JavaScript
- Canvas-based network visualizations
